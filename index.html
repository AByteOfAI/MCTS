02 ‚Äî The Algorithm

How MCTS Works

MCTS builds a decision tree incrementally through four iterative stages, balancing exploration of unknown paths with exploitation of known high-reward actions.

Stage 1

Selection

Traverse the tree from root to a leaf node using the UCB formula to select the most promising branch.

‚Üí
Stage 2

Expansion

Add new child nodes at the leaf to represent unexplored actions and potential future states.

‚Üí
Stage 3

Simulation

Run rollouts from new nodes to estimate future rewards. DRL replaces random rollouts for better accuracy.

‚Üí
Stage 4

Backpropagation

Update node statistics along the selected path, reinforcing high-reward routes for future decisions.
